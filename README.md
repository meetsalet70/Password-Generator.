# 🔐 Password Generator

A simple and responsive Password Generator built using **HTML, CSS, and JavaScript**. This project generates strong and secure passwords based on user-selected options.

## 🚀 Features

- Generate secure random passwords
- Select password length (4–30 characters)
- Include Uppercase letters (A-Z)
- Include Lowercase letters (a-z)
- Include Numbers (0-9)
- Include Special Symbols (!@#$%^&*)
- One-click Copy Password button
- Responsive and user-friendly interface

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript

## ▶️ How to Run

1. Download or clone this repository.
2. Open the `index.html` file in your web browser.
3. Select the password options.
4. Click **Generate Password**.
5. Copy and use your secure password.
